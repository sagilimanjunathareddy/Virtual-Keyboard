Gesture Virtual Keyboard
Overview
A virtual keyboard controlled by hand gestures captured via webcam. Users can type by pinching fingers over keys displayed on screen. Supports multi-hand detection, caps lock, shift, backspace, and enter keys.

Features
Laptop-style QWERTY keyboard layout

Detects index finger and thumb pinch for key press

Multi-hand support for typing with both hands

Caps Lock and Shift functionality

Backspace, Space, and Enter keys included

Adjustable webcam resolution

Visual feedback on hovered and pressed keys

Tools and Libraries
Python 3.11

OpenCV (for webcam capture and UI rendering)

MediaPipe (for hand and finger tracking)

Pygame
